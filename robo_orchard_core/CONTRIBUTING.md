# Contribution Guide

## Install by editable mode

```bash
make install-editable
```

## Install development requirements

```bash
make dev-env
```

## Lint

```bash
make check-lint
```

## Auto format

```bash
make auto-format
```

### Type checking

```bash
pyright
```

## Build docs

```bash
make doc
```

## Run test

```bash
make test
```
